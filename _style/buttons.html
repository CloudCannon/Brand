--- title: Buttons ---

Regular

This should be used for unimportant actions.

Important

This should be used for form submissions and required actions.

Warning

This should be used for dangerous/reversable actions.

Danger

This should be used for dangerous/irreversable actions.

Sizes

The size of the button is important. For multiple actions in a list the small button should be used. The large button should be used primarily on call to actions in the page hero. Block buttons should always be used for forms in the app.

Other

These are generally not used.
